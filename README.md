# Personalized AI Tutor Web Application

## Project Overview

The **Personalized AI Tutor Web Application** is an intelligent web-based learning system developed using **Python, Flask, HTML, CSS, JavaScript, Bootstrap, and Machine Learning**. The application predicts a student's academic performance based on various educational factors and provides personalized learning recommendations to help improve their studies.

The system uses a trained Machine Learning model to analyze student data such as attendance, GPA, study hours, quiz scores, assignment completion, and subject marks. Based on these inputs, it predicts the student's performance level and displays customized feedback.

This project demonstrates the integration of web development with Artificial Intelligence to create an interactive educational platform.

# Machine Learning Workflow

1. Generate the student dataset.
2. Train the Machine Learning model.
3. Save the trained model.
4. Load the model in Flask.
5. Accept student input.
6. Predict student performance.
7. Display personalized recommendations.
